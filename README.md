### Sobre o Problema

Nosso projeto é na área da Astronomia. Utilizamos dados do Sloan Digital Sky Survey (SDSS), um dos maiores projetos de mapeamento astronômico da atualidade, que foi realizado a partir de um telescópio digital no Novo México, Estados Unidos.

A classificação manual de objetos celestiais, como estrelas, galáxias e quasares, é extremamente demorada e praticamente inviável, devido ao alto volume e grande complexidade dos dados.

Nosso projeto visa automatizar esse processo de classificação utilizando algorítimos de classificação.


### Pergunta de Negócio

**Pergunta:** É possível classificar de forma automática se um objeto é uma estrela, uma galáxia ou um quasar, a partir de suas características observadas durante o projeto SDSS?
